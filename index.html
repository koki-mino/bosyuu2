このページについて

このページは、Googleスプレッドシートに登録した 「中学生ボランティア募集情報」を自動で読み込んで一覧表示します。
スプレッドシートを更新すると、ページの表示も自動で更新されます。

スプレッドシートの列名ルール（管理者向け）
  • title … イベント名
  • date … 日付（例：2025-11-23 や 11/23（日））
  • time … 時間（例：9:00〜12:00）
  • place … 場所
  • target … 対象（例：中1〜中3）
  • category … 分野（例：子ども・環境・イベント など）
  • capacity … 定員（例：20人・先着順 など）
  • deadline … 申込締切
  • description … 内容（簡単な説明）
  • apply_url … 申込フォームURL（Googleフォームなど）
